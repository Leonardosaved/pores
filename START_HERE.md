---

### OPTION B: Create Professional Installer (Recommended)

1. **Install NSIS** (one-time, 10 minutes):
   - Go to: https://nsis.sourceforge.io/
   - Download and install
   - Accept all defaults

2. **Build the installer** (2 minutes):
   - Open Command Prompt
   - Navigate to your ROI Analyzer folder
   - Run: `makensis installer.nsi`

3. **Get the installer**:
   - File created: `ROI_Analyzer_Installer.exe`
   - **This is what you share** with friends

4. **They run the installer**:
   - Double-click `ROI_Analyzer_Installer.exe`
   - Follow the wizard
   - Shortcuts created automatically
   - They can launch from Start Menu or Desktop
